# VOILA

Brief description

## Dependencies
- cmake >= 3.11
- bison >= 3.2
- RE/flex
- gcc >= 9 / clang >= 8
- ninja
- llvm
- Doxygen


## Setup
For setup run:

```
./configure
```

The configuration script invokes cmake, creates a subdirectory called `build` and configures the project. To build the project:
```
cd build
ninja
```
